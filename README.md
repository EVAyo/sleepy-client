## 介绍

### 概括
本项目是为了改善 [sleepy](https://github.com/wyf9/sleepy) 中客户端糟糕的体验诞生的。提供了更加美观的界面，更加友好的操作体验。特性请详见[此处](#特性)。

### 特性

- 精美的 Fluent Design 风格界面
- 简介明了的设置
- 更友好且方便的操作体验
- 错误处理
- 可自定义扩展（）
